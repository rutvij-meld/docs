---
title: "Testing Sell, Transfer, NFT Checkout"
---

The Buy flow can be tested in service provider's sandboxes, because the user is able to complete a test transaction and get back data as if it was a real transaction. However, it is not possible to test the sell, transfer, or NFT checkout flows end to end in service provider sandboxes.

In a service provider sandbox you can get a sell quote, as well as launch the widget for a sell, transfer, or NFT checkout, however you cannot complete the transaction. To test this flows you will have to make a real transaction with real crypto / fiat. That is why Meld recommends using the buy flow for most of your initial testing.
