[Onramp.Money](https://onramp.money/) is an onramp / offramp that specifically targets India as its market. As a result, it supports local payment methods such as UPI and IPMS that have high conversion in India.

# Sandbox Testing

In sandbox, for the crypto onramp flow, Onramp.Money only supports USDT_POLYGONTEST. Additionally, to get past the phone number screen, you can use the phone number +91 8660031402 and the passcode 112233. Lastly, your appId must be set to 2, as it's the official test appId.

# Configs

If you have your own Onramp.Money account that you wish to use with Meld, Meld will need your API Key, API Secret, and App Id.

![](https://files.readme.io/e738f08-image.png)

The AppId is essentially a unique identifier for your account that is supplied by Onramp.Money. Meld passes your appId onto Onramp.Money in the url so that Onramp.Money knows whose account is making the call.
