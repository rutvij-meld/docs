# Introduction

The Meld Crypto Stack allows you to offer a comprehensive and straightforward way to connect to multiple on-ramps and off-ramps and crypto flows for your Customers. Using our API, you can provide your customers with quotes and use our widget to initiate the purchase process. After you review the documentation, read the API Reference for the Crypto Stack to understand what is required to make the API calls.

To onboard with Meld for Crypto, first follow the steps found in the [onboarding guide](https://docs.meld.io/docs/onboarding) then come back to this page.

# [Integration Options](https://docs.meld.io/docs/integration-options)

There are 2 main ways you can use Meld's crypto products, either through the white label widget experience or the all in one wizard. Both options support buying and selling crypto, however at the moment only the white label widget experience supports transferring crypto and NFT checkouts. The key difference is the wizard relies on your UI to gather and pass in some information while the wizard can function as a standalone UI for the entire crypto flow, but then therefore has less customization options, since it is Meld's cobrandable UI.

Reasons to use Meld's White Label Widget Experience:

* You want to use your own UI to gather information about the transaction and show quotes to the user.
* You want flexibility and control over the customization of the UI as well as the flow.

Reasons to use Meld's All in One Wizard:

* You want an all-in-one UI you can launch immediately to go to market asap.

Read more about these options [here](https://docs.meld.io/docs/integration-options).
