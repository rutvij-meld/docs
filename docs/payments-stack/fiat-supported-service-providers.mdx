---
title: "Supported Service Providers"
---

This page lists the fiat payment service providers Meld supports and the capabilities each one offers — card payments, refunds, credential passthrough, external refunds, tokenization, authorization and capture, and ACH payments. Use it to pick the right provider for your integration and to confirm which features you can rely on.

<Note>
An `X` in a cell means the provider supports that capability through Meld. An empty cell means it does not.
</Note>

<Note>
**Resources**

- Certain service providers have features that are unique. For example, Checkout.com allows developers to add a custom billing descriptor to a transaction. We expose service-provider-specific functionality in the `serviceProviderDetails` object. Details and parameters can be found in the Transactions endpoint of the API reference.
- For credential passthrough headers per provider, see [Credential Passthrough](/docs/payments-stack/credential-passthrough).
</Note>
