---
title: "Payments Use Cases"
---

There are several payments use cases that can be enabled through Meld.

1. Cards - Meld supports charging users via card information through multiple providers. 
2. ACH - Meld supports ACH payin and payout transactions through multiple providers. See [here](https://docs.meld.io/docs/making-ach-payments) for more information. Before initiating an ACH transaction, you will need to [verify the user's bank account](https://docs.meld.io/docs/account-verification) using Meld's [Bank Linking Stack](https://docs.meld.io/docs/bank-linking-quickstart). This allows you to lower fraud and errors, and is the industry standard methodology to get user permissioned access to the user's account number, routing number (using the IDENTIFIERS product), and name (using the OWNERS product).
