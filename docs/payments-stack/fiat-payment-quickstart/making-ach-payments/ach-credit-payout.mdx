---
title: "ACH Payout (Credit)"
---

ACH Payouts are any transaction where money flows from the merchant to the buyer.

Make sure you have a [Meld customerId](https://docs.meld.io/docs/creating-a-payments-customer) for the buyer ready before continuing. Sample API calls for all the steps for making a payin are found on this page.

# Make an ACH Credit (aka Payout) transaction

1. Call Meld's [/transactions/payout endpoint](https://docs.meld.io/reference/payouts-transactions-create) in order to make an ACH Payout. Make sure to pass in the customerId of the buyer you created in the [Creating a Customer](https://docs.meld.io/docs/creating-a-payments-customer) stage. This is where you will pass in the ACH details including the account type, account number and routing number. 
2. Once the transaction has been submitted, listen for the [TRANSACTION\_FIAT\_CREATED webhook](https://docs.meld.io/docs/webhook-events-fiat-payment#transaction_fiat_created) to know that a transaction has been initiated. You can use this webhook as a trigger to call Meld's [get transaction endpoint](https://docs.meld.io/reference/payments-transactions-get) to fetch the latest transaction details.
3. Depending on the underlying service provider, you may receive a [TRANSACTION\_FIAT\_UPDATED webhook](https://docs.meld.io/docs/webhook-events-fiat-payment#transaction_fiat_updated) letting you know if the transaction has succeeded, or failed, or if something else has occurred. Use this webhook as a trigger to fetch the latest transaction details. For certain providers who do not support webhooks, you will not receive updates after the transaction is submitted, however, if the transaction fails upon submission you will see that through response object immediately.
   > 📘 Webhook Support
   >
   > To know which providers support webhooks and which don't, see our provider [testing guide](https://docs.meld.io/docs/ach-testing).

# API Calls

## Make a Payout

Here is a sample call to [/transactions/payout](https://docs.meld.io/v0.0/reference/payouts-transactions-create) to make an ACH Payout transaction:

```json
{
    "customerId": "WXDeHGgNJ1t1nFnoVY43Bh",
    "sourceAmount": "15.00",
    "sourceCurrencyCode": "USD",
    "paymentMethod" : {
        "type" : "ACH",
        "details": {
            "accountType" : "CHECKING", // should be CHECKING, SAVINGS, or BUSINESS_CHECKING
            "routingNumber": "987654321",
            "accountNumber": "43214321"
        }
    },
    "serviceProvider": "MOOV"
}
```
