---
title: "Error responses"
---

When a request fails, Meld endpoints return a response that includes the errors fields in the body. The following table lists the error code you might receive from our API. 

The error message includes both Meld's error code and the service provider's error details. Reference [Error schema](https://docs.meld.io/docs/error-schema).

# General

<Table align={["left","left"]}>
  <thead>
    <tr>
      <th>
        Error code
      </th>

      <th>
        Error message
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        `UNKNOWN`
      </td>

      <td>
        Unknown Error
      </td>
    </tr>

    <tr>
      <td>
        `BAD_REQUEST`
      </td>

      <td>
        Bad request

        * \*Note:\*\* When a bad request is received, the system will return a user-friendly representation of the error, such as:
          <li>[amount] Must be a decimal value greater than zero</li>
          <li>[serviceProvider] Must be a service provider</li>
          <li>[serviceTransactionId] must not be blank</li>
      </td>
    </tr>
  </tbody>
</Table>
