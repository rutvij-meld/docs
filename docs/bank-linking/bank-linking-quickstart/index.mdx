# Introduction

Welcome to Meld's Bank Linking Product! Meld’s Bank Linking stack enables you to work with multiple bank linking providers through an integrated API and UX flow.

Before you start using Meld's products, first make sure that you have completed [Onboarding](https://docs.meld.io/docs/onboarding) and have your Meld API key available.

The page includes a summary of how to create and manage a connection, a list of Meld environments, and a glossary with an accompanying diagram to explain Meld's terminology.

# Accounts with Service Providers

Meld has reseller agreements with Finicity, MX, and Akoya, and can create an account for you with each of these providers. For the other providers, if you don't have your own account with the provider already, Meld can temporatily share their account with you for testing while your account is being spun up. Contact Meld for more details on this.
