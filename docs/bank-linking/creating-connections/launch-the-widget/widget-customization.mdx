---
title: "Widget Customization"
---

You can customize several UI elements of the Meld Connect Widget from the Meld dashboard's Developer tab. This page is for developers and PMs who want the widget to match their product's branding. Customization happens in the dashboard — no code changes are required.

The following options are configurable:

* **Font** -- This is the font shown for all text within the Meld picker

* **Institution Select Text** -- This is the text at the top of the Meld picker

![](https://files.readme.io/c949e2c-image.png)

These values are prepopulated with defaults; press **Reset** to restore them.

You can also customize the colors of the Routing Retry buttons.

<Note>
To customize Routing Retry button colors, contact Meld support — this is currently a Meld-assisted configuration rather than a self-serve setting.
</Note>
