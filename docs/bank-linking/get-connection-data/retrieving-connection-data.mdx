Once your customer has successfully completed a connection, you can access their financial information.

# Accessing customer financial data

In order to access your customer's financial data, you can call the endpoints below. We typically adhere to two endpoint patterns:   

* Get a singular result by `id` 
* Filter through all results by various query parameters. Some of these are described below:
  * `externalCustomerId` - the id you provided Meld for your customer.
  * `customerId` - the id Meld assigns for your customer.
  * `institutionId` - the Meld id for a financial institution.
  * `financialAccountId` - the Meld id for your customer's financial account.

## Financial Data

To get back information on [Balances](https://docs.meld.io/docs/account-balances), [Identifiers](https://docs.meld.io/docs/account-identifiers), or [Owners](https://docs.meld.io/docs/account-owners), use the [financial accounts endpoint](https://docs.meld.io/reference/bank-linking-accounts-search). See [here](https://docs.meld.io/docs/get-connection-data#get-balances-owners-and-identifiers-data) for an example.

To get back a list of [Transactions](https://docs.meld.io/docs/account-transactions), use the [transactions endpoint](https://docs.meld.io/reference/bank-linking-transactions-search). See [here](https://docs.meld.io/docs/get-connection-data#get-transactions-data) for an example. Note that transactions are ordered by newest first, per each financial account. For more details on Meld's recommendation on how to fetch financial transactions, see [here](https://docs.meld.io/docs/fetching-transaction-information).

To get back a list of [Investment Holdings](https://docs.meld.io/docs/account-investments#investment-holdings), use the [investment holdings endpoint](https://docs.meld.io/reference/bank-linking-investment-holdings-search). See [here](https://docs.meld.io/docs/get-connection-data#get-investments-data) for an example.

To get back a list of [Investment Transactions](https://docs.meld.io/docs/account-investments#investment-transactions), use the [investment transactions endpoint](https://docs.meld.io/reference/bank-linking-investment-transactions-search). See [here](https://docs.meld.io/docs/get-connection-data#get-investments-data) for an example.

## Metadata

To get back a list of connections, use the [connections endpoint](https://docs.meld.io/reference/bank-linking-connections-search).
