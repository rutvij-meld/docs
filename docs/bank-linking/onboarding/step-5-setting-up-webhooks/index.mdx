---
title: "Step 4: Setting Up Webhooks"
---

Setting up webhooks is a 2 step process.

Step 1: In order for Meld to receive webhooks from the providers you have configured in step 2, you must set up these webhooks for each service provider. Instructions can be found [here](https://docs.meld.io/docs/webhook-quickstart#steps-to-send-webhooks-from-the-service-provider-to-meld).

Step 2: If you would like to receive webhooks with updates from Meld, go to the **Developer** tab and the **Webhooks** subtab, and set up webhooks as described [here](https://docs.meld.io/docs/webhook-quickstart#steps-to-receive-webhooks-from-meld). 

![](https://files.readme.io/f66d007-image.png)

Step 1 of webhook setup is necessary for Meld to track your transactions / connections. Step 2 of webhook setup is optional, as you can use the **Transactions** tab to view your completed transactions and connections.
