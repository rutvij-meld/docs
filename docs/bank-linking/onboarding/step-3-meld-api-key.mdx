# Not Using the Crypto Wizard

This section assumes that you are not using Meld's crypto wizard. If you are, see [here](https://docs.meld.io/docs/step-3-meld-api-key#using-the-crypto-wizard) instead.

After adding your integrations, go to the **Developer** tab and the **API Keys** subtab to grab your API key, which is automatically generated and includes the credentials for all of the integrations you added in the previous step.

![](https://files.readme.io/9035875-image.png)

Keys named **Production** and **Sandbox** are private keys you will use to call Meld's API. **Production** keys are in Meld's production environment, and **Sandbox** keys are in Meld's sandbox environment. Remember to put the word `BASIC` in front of the key when making an API call.

Unless you are using the **Public Crypto Wizard** key, please treat all API keys as absolutely confidential! If your API key is ever exposed, please inform Meld immediately so that Meld can roll your key.

# Using the Crypto Wizard

If you are only using the [Meld Crypto Wizard](https://docs.meld.io/docs/crypto-wizard), ignore any key marked **Production** or **Sandbox**. Instead, you will see a **Public Crypto Wizard** key as well, and the url in the popup is the url for your personal crypto wizard. Do not use any other API key for your crypto wizard or you may risk exposing sensitive data. This only applies to the Crypto Stack, not the Bank Linking or Payments Stacks.
