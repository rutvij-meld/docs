# Dashboard Data Management

The Meld dashboard provides transaction monitoring and analytics to help you track and optimize your integration performance.

## Transaction Overview

View all transactions from your Meld integrations in the **Transactions** tab. Transactions are displayed with the most recent first, showing:

* Transaction status and completion details
* Service provider used
* Transaction amounts
* Timestamps

### Filtering Options

Filter your transaction data by:

* **Date Range**: Specific time periods
* **Service Provider**: Individual payment providers
* **Transaction Status**: Completed, failed, or pending

![Transaction Dashboard](https://files.readme.io/2c1ba3a-image.png)

### Data Export

Export transaction data as CSV files for:

* Local storage and backup
* Custom analysis and reporting
* Compliance and record-keeping

## Analytics Dashboard

The **Analytics** tab provides performance insights to optimize your integration:

### Key Metrics

* **Conversion Rates**: Success rates by provider and payment method
* **Popular Providers**: Which services your users prefer
* **Market Performance**: Geographic and currency trends
* **Token Analysis**: Most traded cryptocurrencies

### Performance Insights

* Provider comparison data
* Peak usage patterns
* Success rate trends

![Analytics Dashboard](https://files.readme.io/806c722-image.png)

## Using Dashboard Data

**For Development:**

* Identify optimization opportunities
* Debug failed transactions

**For Business:**

* Track user preferences and behavior
* Optimize provider selection
* Monitor conversion improvements
