# Sell Flow - Crypto Offramp

Sell Flow, also known as offramping, allows users to sell cryptocurrency and receive fiat currency through their preferred payment method.

## How the Sell Flow Works

1. **User Input:** Amount of crypto to sell, desired fiat currency, payout method
2. **Quote Comparison:** Multiple provider quotes with rates and fees
3. **Provider Selection:** User chooses best quote
4. **KYC / KYB:** The user must prove their identify, once per onramp. In the Ramp flow this is via the onramp's UI, in the Virtual Account flow this will be manually processed by Meld for the time being.
5. **Crypto Transfer:** User sends crypto to onramp's wallet
6. **Fiat Payout:** Provider sends fiat to user's bank account or card
