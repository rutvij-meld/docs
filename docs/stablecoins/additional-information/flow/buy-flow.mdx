# Buy Flow - Crypto Onramp

Buy Flow, also known as onramping, allows users to purchase cryptocurrency using fiat currency and receive it directly in their wallet.

## How the Buy Flow Works

1. **User Input:** Amount to spend, preferred cryptocurrency, payment method, fiat currency, wallet address
2. **Quote Comparison:** Multiple provider quotes with rates and fees
3. **Provider Selection:** User chooses best quote
4. **KYC / KYB:** The user must prove their identify, once per onramp. In the Ramp flow this is via the onramp's UI, in the Virtual Account flow this will be manually processed by Meld for the time being.
5. **Payment:** User completes payment. In the Ramp flow this is via the onramp's UI, in the Virtual Account flow this is from the user's bank app.
6. **Crypto Delivery:** Cryptocurrency sent directly to user's wallet
