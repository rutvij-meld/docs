title: "Product 2: Meld Checkout"
description: "Overview of the Meld Checkout Flow"
---

The Meld Checkout flow gives you a custom checkout url that you can embed or redirect to from your app. This UI will allow users to buy or sell crypto. The actual transaction, including KYC and payment, occurs within the onramp's UI. Setting up the Meld Checkout flow takes minutes.

1. For the 5 minute quickstart guide to execute the flow for the first time, click [here](https://docs.meld.io/docs/meld-checkout-quickstart).
2. Once you have completed the quickstart, click [here](https://docs.meld.io/docs/meld-checkout-guide) for the full end to end integration instructions.

If you are using the Meld checkout, you will not need to use any of Meld's APIs, therefore you can skip viewing the Meld API Reference.
