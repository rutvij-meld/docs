title: "Product 2: Meld Checkout"
description: "Overview of the Meld Checkout Flow"
---

Meld Checkout is for product and engineering teams who want a hosted, ready-to-use crypto buy/sell UI they can embed or redirect to from their app. The hosted UI collects the user's intent, and the actual transaction — including KYC and payment — happens within the onramp's UI. Setting up the Meld Checkout flow takes minutes.

1. For the 5-minute quickstart that runs the flow end-to-end, see the [Meld Checkout Quickstart](/docs/stablecoins/meld-checkout-integration/meld-checkout-quickstart).
2. Once the quickstart works, see the [Meld Checkout Guide](/docs/stablecoins/meld-checkout-integration/meld-checkout-guide) for the full integration walkthrough, URL parameters, and styling.

<Tip>
Meld Checkout requires no backend API calls — you only need a Public Crypto Widget Key from the Meld dashboard.
</Tip>

<Note>
The Meld Checkout URL is enabled on request. If your dashboard does not show a Public Crypto Widget Key, contact your account manager.
</Note>
