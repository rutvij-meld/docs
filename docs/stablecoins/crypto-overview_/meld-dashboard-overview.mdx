The Meld Dashboard is the web UI Meld customers use to monitor transactions, configure their integration, and manage API credentials. This page walks new dashboard users through each section so they know where to go for which task.

## Getting access

After receiving your dashboard invite link, create your dashboard password to activate access.

* **Dashboard URL:** [https://dashboard.meld.io/#/sign-in](https://dashboard.meld.io/#/sign-in)

<Note>
Sandbox and production environments each have a separate dashboard login. If you have access to both, make sure you're signed into the right one before reading transaction data or copying keys.
</Note>

## What you can do in the dashboard

* **Transactions** — Monitor every transaction processed through your account, filter by status, and inspect individual transaction details.
* **Network Partners** — In the Integration tab, see which ramp and network partners are enabled for your account.
* **Analytics** — View insights and performance data for transactions and supported markets.
* **Developer**
  * Reveal and rotate **API Keys**
  * Configure **Webhook endpoints** and subscribed events
  * Copy your **Public Crypto Widget Key** URL for the Meld Checkout flow
* **Manage Users** — Add teammates and control access to the dashboard.

<Tip>
The Meld Checkout URL is only available on request. Contact your account manager to enable it for your account.
</Tip>

This dashboard helps you monitor activity, manage integrations, and control your account efficiently.
