Meld offers three digital asset products that help fintechs, wallets, and apps let their users buy, sell, and transfer crypto globally with a single integration. Pick the product that matches how much UI control and which payment methods you need, then follow the matching integration guide.

## Three products at a glance

1. [White-Label API](#white-label-api) — you build the UI, Meld powers the rails
2. [Meld Checkout](#meld-checkout) — Meld's hosted UI, fastest path to launch
3. [Virtual Account](#virtual-account) — bank-transfer onramp / offramp with no provider UI

<Tip>
New to Meld? Start with the [Quickstart](/docs/stablecoins/crypto-overview_/quickstart) to make your first API call in 5 minutes.
</Tip>

## Getting Started with Meld

Meld helps your users buy, sell, and transfer crypto around the world with a single integration. To learn more about Meld, visit [meld.io](https://www.meld.io/).

Many of our customers have already integrated one or two onramps, but realized that they would need many more for global coverage and redundancy. That's why you come to Meld: one integration, global coverage, competitive pricing, and smart routing of users to the best onramp for them.

## 🚀 Choose the right product

### White Label API

**Choose the [White-Label API flow](/docs/stablecoins/white-label-api-integration) if:**

* You want to build your own UI to gather info from the user on what kind of transaction they want to make. Phantom Wallet, Uniswap Wallet, and MetaMask Wallet all use this product to power their onramping
* You are fine with your users completing the transaction within the onramp's UI
* You want your users to be able to buy, sell, or transfer tokens globally using onramps
* You want your users to be able to use any payment method, not just bank transfer

### Meld Checkout

**Choose the [Meld Checkout flow](/docs/stablecoins/meld-checkout-integration) if:**

* You want to go to market quickly
* You are willing to use Meld's UI to gather info from the user on what kind of transaction they want to make, and then direct them to the onramp's UI to complete the transaction
* You want your users to be able to buy or sell tokens globally using onramps, but don't need to support transfers
* You want your users to be able to use any payment method, not just bank transfer

<Note>
The Meld Checkout URL is only available on request. Please contact your account manager to enable it.
</Note>

### Virtual Account

**Choose the [Virtual Account flow](/docs/stablecoins/virtual-account-integration) if:**

* You want the onramp to create a virtual bank account for each of your customers
* You want to build your own UI to facilitate the transaction
* You don't want your users to leave your UI and go to the onramp's UI
* You want yourself and/or your users to be able to buy or sell tokens globally, but don't need to support transfers
* You are fine with your customers paying only with bank transfer

## 🔄 Comparison

|                               | White-Label API                 | Meld Checkout                                        | Virtual Account                                |
| :---------------------------- | :------------------------------ | :--------------------------------------------------- | :--------------------------------------------- |
| `Integration time`            | ~1 week                         | ~15 mins                                             | ~1 week                                        |
| `UI flow`                     | Your UI → Onramp UI → Your UI   | Your UI → Meld UI → Onramp UI → Your UI              | Your UI → User's bank or wallet UI → Your UI   |
| `Transaction types`           | Buy, Sell, Transfer             | Buy, Sell                                            | Buy, Sell                                      |
| `Payment methods`             | All                             | All                                                  | Bank transfer only                             |
| `How it works`                | Call Meld's APIs                | Get URL from Meld to embed / redirect to in your app | Call Meld's APIs                               |

Once you decide which Meld product fits your use case, head to the matching integration guide above to start building.
