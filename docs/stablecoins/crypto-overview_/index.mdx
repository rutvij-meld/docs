# Getting Started with Meld

Meld helps your users buy, sell, and transfer crypto around the world with a single integration. To learn more about Meld, visit [meld.io](https://www.meld.io/) .

Many of our customers have already integrated one of several onramps, but realized that they would need to integrate many more for global coverage and redundancy. That's why you come to Meld: one integration, global coverage, great pricing, and routing your users to the best onramp for them.

## 🚀 Getting Started

First, figure out which Meld crypto product(s) are most relevant for your use case.

### White Label API

**Choose the [White-Label API flow](https://docs.meld.io/docs/white-label-api-integration) if**

* **You want to build your own UI to gather info from the user on what kind of transaction they want to make**. Phantom Wallet, Uniswap Wallet, and Metamask Wallet all use this product to power their onramping
* You are fine with your users completing the transaction within the onramps' UI
* You want your users to be able to buy, sell, or transfer tokens globally using onramps
* You want your users to be able to use any payment method they want, not just bank transfer

### Meld Checkout

**Choose the [Meld Checkout flow](https://docs.meld.io/docs/meld-checkout-integration) if**

* **You want to go to market quickly**
* Meld checkout URL is only available on request. ** Please connect your account manager to Add**
* You are willing to use Meld's UI to gather info from the user on what kind of transaction they want to make, and then be directed to the onramps' UI to actually complete the transaction
* You want your users to be able to buy or sell tokens globally using onramps, but don't need to support transfers
* You want your users to be able to use any payment method they want, not just bank transfer

### Virtual Account

**Choose the [Virtual Account flow](https://docs.meld.io/docs/virtual-account-integration) if**

* **You want the onramp to create a virtual bank account for each of your customers**
* You want to build your own UI to facilitate the transaction
* You don't want your user to leave your UI and go to the onramps' UI
* You want yourself and/or your users to be able to buy or sell tokens globally using onramps, but don't need to support transfers
* You are fine with your customers paying only with bank transfer

|                               | White Label API                 | Meld Checkout                                        | Virtual Account                                |
| :---------------------------- | :------------------------------ | :--------------------------------------------------- | :--------------------------------------------- |
| `Integration time`            | 1 week                          | 15 mins                                              | 1 week                                         |
| `UI Flow`                     | Your UI -> Onramp UI -> Your UI | Your UI -> Meld UI -> Onramp UI -> Your UI           | Your UI -> User's bank or wallet UI -> Your UI |
| `Transaction Types Supported` | Buy, Sell, Transfer             | Buy, Sell                                            | Buy, Sell                                      |
| `Payment Methods Supported`   | All                             | All                                                  | Bank Transfer Only                             |
| `How It Works`                | Call Meld's APIs                | Get url from Meld to embed / redirect to in your app | Call Meld's APIs                               |

Once you have decided which Meld product best suites your use case, then you can quickly integrate with Meld.
