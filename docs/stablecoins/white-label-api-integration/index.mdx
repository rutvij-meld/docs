title: "Product 1: White-Label API"
description: "Overview of the White-Label API Integration Flow"
---

The white-label API integration allows you to build a UI on top of Meld's APIs that enables your users to buy, sell, or transfer crypto. The actual transaction, including KYC and payment, occurs within the onramp's UI. This is the most popular choice due to UI flexibility.

1. For the 30 minute quickstart guide to execute the flow for the first time, click [here](https://docs.meld.io/docs/whitelabel-quickstart).
2. Once you have completed the quickstart, click [here](https://docs.meld.io/docs/whitelabel-api-guide) for the full end to end integration instructions.
