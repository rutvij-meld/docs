## Meld Tokens

Whenever you are calling a Meld endpoint about a particular token, make sure you are using the Meld token name. Meld follows the TOKEN\_NETWORK naming convention, for example Ethereum on Polygon would be ETH\_POLYGON.

Service Providers accessed through Meld support a wide variety of cryptocurrencies. You can find a list of supported cryptocurrencies for onramps and offramps below: 

[Supported Cryptocurrencies for Onramps](#supported-cryptocurrencies-for-onramps)\
[Supported Cryptocurrencies for Offramps](#supported-cryptocurrencies-for-offramps)

The `GET /service-providers/properties/crypto-currencies` endpoint can also be used to obtain this information.


<Note>
Interacting with our Coverage docs

Please note that all coverage sheets are set to 'view only' access. To filter the data and use these sheets more easily, please create a local filter.

To create a local filter, click on 'Data' > 'Filter views' > 'Create new temporary filter view'. Now, you can interact with Meld's coverage documents with ease!
</Note>
