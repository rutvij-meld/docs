<Tip>
Service Provider Properties API Access

The data on supported Crypto Currencies, Countries, Fiat Currencies, and Payment Methods is also available via API through our [Service Provider Properties](https://docs.meld.io/reference/service-provider#/) endpoints.

The Service Provider Properties endpoints are a powerful and normalized way for developers to assess which service providers provide the services desired by a Customer based on the token to be purchased, available payment methods, and supported countries and currencies.
</Tip>
